### Code Blocks

Here's a JavaScript code block:

```javascript
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

const result = fibonacci(10);
console.log(result); // Output: 55
```

Python example with syntax highlighting:

```python
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

numbers = [3, 6, 8, 10, 1, 2, 1]
print(quicksort(numbers))
```

## Math Equations

### Inline Math

You can write inline math like $E = mc^2$ or $a^2 + b^2 = c^2$ directly in your text. The Pythagorean theorem states that $\sqrt{a^2 + b^2} = c$. 

More examples: $\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$ and the famous $e^{i\pi} + 1 = 0$.

### Block Math Equations

Here's the quadratic formula displayed as a block equation:

$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

Maxwell's equations in differential form:

$$
\begin{aligned}
\nabla \cdot \mathbf{E} &= \frac{\rho}{\epsilon_0} \\
\nabla \cdot \mathbf{B} &= 0 \\
\nabla \times \mathbf{E} &= -\frac{\partial \mathbf{B}}{\partial t} \\
\nabla \times \mathbf{B} &= \mu_0\mathbf{J} + \mu_0\epsilon_0\frac{\partial \mathbf{E}}{\partial t}
\end{aligned}
$$

### Matrix Notation

The identity matrix:

$$
I = \begin{pmatrix}
1 & 0 & 0 \\
0 & 1 & 0 \\
0 & 0 & 1
\end{pmatrix}
$$

### Summations and Products

The definition of the arithmetic mean:

$$
\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i
$$

The factorial function:

$$
n! = \prod_{i=1}^{n} i
$$

### Calculus

The fundamental theorem of calculus:

$$
\int_a^b f(x)\,dx = F(b) - F(a)
$$

Taylor series expansion:

$$
f(x) = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \frac{f'''(a)}{3!}(x-a)^3 + \cdots
$$

### Probability and Statistics

Bayes' theorem:

$$
P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}
$$

Normal distribution:

$$
f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2}
$$

### Limits

$$
\lim_{x \to \infty} \frac{1}{x} = 0
$$

$$
\lim_{n \to \infty} \left(1 + \frac{1}{n}\right)^n = e
$$

### Set Theory

$$
A \cup B = \{x : x \in A \text{ or } x \in B\}
$$

$$
A \cap B = \{x : x \in A \text{ and } x \in B\}
$$

### Greek Letters and Special Symbols

$$
\alpha, \beta, \gamma, \delta, \epsilon, \zeta, \eta, \theta, \iota, \kappa, \lambda, \mu
$$

$$
\Gamma, \Delta, \Theta, \Lambda, \Xi, \Pi, \Sigma, \Upsilon, \Phi, \Psi, \Omega

### Code with Math Explanation

Here's an implementation of the quadratic formula $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$:

```javascript
function solveQuadratic(a, b, c) {
  const discriminant = b * b - 4 * a * c;
  
  if (discriminant < 0) {
    return []; // No real solutions
  }
  
  const sqrtDisc = Math.sqrt(discriminant);
  const x1 = (-b + sqrtDisc) / (2 * a);
  const x2 = (-b - sqrtDisc) / (2 * a);
  
  return discriminant === 0 ? [x1] : [x1, x2];
}

// Example: x² - 5x + 6 = 0
console.log(solveQuadratic(1, -5, 6)); // [3, 2]
```

## Footnotes

Footnotes allow you to add references and additional information without cluttering the main text[^1]. They're particularly useful for academic writing and technical documentation[^2].

You can reference the same footnote multiple times[^1], and footnotes can contain **formatted text** and `code snippets`[^3].

[^1]: This is a simple footnote. Footnotes are automatically numbered and linked.

[^2]: MDX with remark-gfm provides full GitHub Flavored Markdown support, including footnotes, tables, task lists, and strikethrough text. Learn more at [GitHub's GFM spec](https://github.github.com/gfm/).

[^3]: Footnotes can include:
    - Multiple paragraphs
    - Code examples: `const x = 10`
    - Even math: $f(x) = x^2$
    
    Just indent the continuation lines with 4 spaces.

## Special Characters and Escaping

You can use special characters like &copy;, &trade;, &reg;, and HTML entities work too: &hearts; &spades; &clubs; &diams;

To escape markdown syntax, use backslashes: \*not italic\*, \[not a link\], \`not code\`.

### Subscript and Superscript

While not standard markdown, you can use HTML: H<sub>2</sub>O for water, and E = mc<sup>2</sup> for Einstein's equation.
